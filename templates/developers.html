For Developers

Cook Roulette provides an outward-facing, REST-ful API for developers interested in implementing random meals into their code. The API is available here:

https://cookroulette.herokuapp.com/api

There are no limits on API calls. Each call returns a JSON object that is of the form:

{"meal": {
---- "vegetable": "tomato",
---- "protein": "bacon",
---- "starch": "macaroni"
---- }
}

where each "meal" skeleton consists of a vegetable, protein, and starch and the ingredients are randomly populated from k-means clustered ingredients.
